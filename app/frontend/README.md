# Frontend Application

## Structure
- `public/` - Static files (HTML, images)
- `src/` - Source code
  - `components/` - React components
  - `pages/` - Page components
  - `services/` - API calls
  - `styles/` - CSS files
  - `App.js` - Main component
  - `index.js` - Entry point
- `package.json` - Dependencies
- `Dockerfile` - Container image
